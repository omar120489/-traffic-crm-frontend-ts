**Status**: ‚úÖ **FULLY IMPLEMENTED** (release-hardened.yml)

**Validation**:
> "Applying the principle of least privilege to workflow permissions greatly reduces exposure if a workflow or job is compromised."  
> ‚Äî [GitHub Token Permissions](https://docs.github.com/en/actions/security-guides/automatic-token-authentication#permissions-for-the-github_token)


**Evidence**: Per-job scoped permissions in hardened workflow
